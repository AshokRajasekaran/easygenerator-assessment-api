## Description

EasyGenerator Assessment API

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment and Security Guide

- Move .env file during build process

- Create rsa keys in seperate folder or as a secret file with high security and read from there to encrypt and validate JWT tokens
